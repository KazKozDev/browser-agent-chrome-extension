# Browser Agent Skills Index (LLM-driven)

Перед выполнением задачи выбери один `skill_id` и следуй его стратегии.
Если ни один не подходит — выбери `general`.

## Доступные скиллы

| ID | Назначение |
|----|------------|
| search | Поиск информации в вебе (google/search/lookup) |
| navigate | Переход на нужную страницу/URL |
| fill_form | Заполнение и отправка форм |
| extract | Извлечение данных/текста со страницы |
| interact | UI-взаимодействия (кнопки, меню, попапы) |
| monitor | Диагностика (console/network) |
| multi_step | Многошаговые/мультисайтовые задачи |
| general | Общая стратегия для прочих задач |
