# Skill: fill_form
Allowed tools: computer, read_page, upload_file, switch_frame, wait_for, done

Prompt:
You are filling a form.
1) Read form structure first.
2) Fill required fields accurately.
3) Use upload_file for file inputs.
4) Use switch_frame if form is inside iframe.
5) Submit and verify result via read_page/wait_for.
6) Call done only after confirmed submit outcome.
