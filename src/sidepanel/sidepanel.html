Settings

Capabilities

Navigation and Actions

Open websites, click elements, type text, submit forms, hover, scroll, and use keyboard shortcuts. Advanced pointer tools are available for right-click, double/triple click, coordinate click, and drag interactions.

Reading and Search

Understand pages with accessibility tree reading, full text extraction, element search by natural language, and Ctrl+F style text search with next/previous match navigation.

Complex Browser Control

Work across tabs and frames, take screenshots (vision-capable providers), upload files to file inputs, and inspect download status.

Safety and Reliability

Sensitive actions require confirmation logic. JavaScript execution is filtered by security rules. If login or CAPTCHA is detected, the agent pauses and waits for manual resume.

Providers and Modes

Choose a model provider in Settings, test connectivity, and switch active tier. The header badge shows the active operating mode/provider profile.
