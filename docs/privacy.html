Privacy Policy

Overview

BrowseAgent is a browser extension that runs entirely in your browser. We do not operate any servers, and we do not collect, store, or transmit your personal data.

Data Collection

BrowseAgent does not collect:

  • Personal information
  • Browsing history
  • Page contents
  • Screenshots
  • API keys
  • Task inputs or outputs

LLM API Communication

When you use BrowseAgent, your task instructions and page context are sent directly from your browser to the LLM provider you configure (e.g., Fireworks AI, xAI, Ollama). This communication happens directly between your browser and the provider's API — no data passes through any BrowseAgent server.

Local Storage

BrowseAgent stores settings, task history, and API keys locally in your browser using Chrome's built-in storage APIs. This data never leaves your device.

Third-Party Services

When you configure integrations (Telegram, Slack, etc.), data is sent directly from your browser to those services. BrowseAgent acts only as the local automation layer.

Open Source

BrowseAgent is open source. You can inspect the full source code to verify these privacy claims.

Contact

For privacy questions, open an issue on our GitHub repository.
