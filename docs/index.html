Skills

Everything this agent does.

Six skill categories covering the full surface of browser automation — from safety to structured extraction.

Browsing & Navigation

Go anywhere on the web

Open any URL, navigate back/forward/reload. Open, list, switch and close tabs — BrowseAgent tabs auto-grouped as "BrowseAgent". Switch between main document and iframes. Restore previously captured page state via snapshots.

Page Interaction

Click, type, scroll, hover

Click with left/right/middle button, single/double/triple. Type into one or multiple fields in a single call. Select dropdown options, hover elements, scroll pages, press keys with modifiers (Ctrl, Shift, Alt, Meta).

Page Reading & Inspection

Accessibility tree + vision

Reads the full accessibility tree with element IDs. Extracts structured data from cards, lists and tables. Natural-language element search, Ctrl+F-style text search. Screenshots with Set-of-Mark overlays for vision-capable models.

Automation & Workflow

Queue, schedule, recover

Plan mode shows steps before executing — approve or cancel. Schedule goals every N minutes/hours via Chrome alarms. Tasks continue in the background when the panel is closed. Interrupted runs resume from persisted checkpoint state.

Safety & Permissions

Injection resistant. Loop protected.

Site blocklist with crypto/payment defaults + custom domains. Tracker and ad blocker via optional DNR ruleset. Login and CAPTCHA detection — agent pauses for manual help. Sensitive-action confirmation required for submit/delete/pay. Prompt injection protection: page content treated as untrusted data.

External Integrations

HTTP requests + notify destinations

http_request — call any REST API directly with no CORS restrictions. notify_connector — send results to connected integrations mid-execution. Eight destinations: Telegram, Slack, Notion, Discord, Airtable, Google Sheets, Email, Webhook.
